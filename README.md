# SUKERU (Image Matte Processor)

AI生成画像（DALL-E 3等）特有の「背景の格子模様」を自動検出し、一括で透過処理を行うツールです。
現場での「実用性」と「時短」を最優先に設計しています。

## 🛠️ 使いかた (Usage)
1. **[images]** フォルダに、透過したい画像を入れます。
2. **[SUKERU.py]** (または exe) を実行します。
3. **[output]** フォルダに、透過された画像が生成されます。

---

## 💡 コンセプト：作業時間を「削る」ための道具
本ツールは、100%完璧な切り抜きを目指すものではなく、「画像編集ソフト（Photoshop等）で仕上げる前の、面倒な下準備を大幅に短縮すること」を目的としています。

* **手間を8割カット**: 手作業で背景を消し始める前の「一番面倒な範囲選択」をAIのロジックで自動化します。
* **仕上げ前提の設計**: キャラクターの境界線にわずかなノイズ（ゴミ）が残る場合がありますが、それは一次処理として割り切ることで、処理速度と実用性を両立させました。細かい調整は使い慣れた画像編集ツールで行ってください。

---

## 🧠 開発のこだわり (Design Philosophy)
私の開発におけるモットーは**「直感的な操作性」**と**「現場への最適化」**です。

* **一貫した命名規則**: YOSHI(Exif削除)やOSERU(PDF印影)といった自作ツール群と同様、PCに詳しくない方でも一目で機能がわかる日本語名をベースにしています。
* **運用重視の設計**: 英語の分からない現場スタッフや、複雑な設定を好まないユーザーのため、UIを排した「配置して実行するだけ」のフローを構築しました。
* **技術的裏付け**: FloodFillアルゴリズムを応用し、被写体のディテールを守りつつ、背景の格子模様を正確に除去するロジックを実装しています。
